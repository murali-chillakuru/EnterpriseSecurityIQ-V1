1. AKS Free-Tier Capacity

WORKAROUND AKSCapacityHeavyUsage in swedencentral

Error: Creating a free tier cluster is unavailable at this time in region swedencentral.

Impact: Container Apps Environment cannot be created in swedencentral.

Workaround: Created the Container Apps Environment in northeurope instead. Adds ~5–10 ms latency per AI Services call.

Resolution: When capacity returns, recreate the environment:

az containerapp env delete -n ESIQNew-env -g ESIQNew-RG --yes
az containerapp env create -n ESIQNew-env -g ESIQNew-RG --location swedencentral `
  --logs-workspace-id (az monitor log-analytics workspace show -g ESIQNew-RG `
    -n ESIQNew-law --query customerId -o tsv)

2. ML Extension Preview Warnings

COSMETIC Experimental class warnings

Symptom: az ml connection create and az ml workspace create emit warnings:

Class DeploymentTemplateOperations: This is an experimental class...
Class AzureAIServicesConnection: This is an experimental class...

Impact: Cosmetic only. Commands succeed.

Resolution: Warnings will disappear as the az ml extension graduates from preview.

3. Container App Scale-to-Zero

PERFORMANCE Cold start latency

Symptom: First request after idle period takes 10–30 seconds.

Impact: Min replicas is 0 to save costs. When all replicas are scaled down, the first request triggers a new container start.

Workaround: Set --min-replicas 1 if cold starts are unacceptable:

az containerapp update -n esiqnew-agent -g ESIQNew-RG --min-replicas 1

Cost Impact: ~$50/month additional for keeping 1 replica always running.

4. Dockerfile Build Context

BUILD COPY failed: file not found

Symptom: az acr build with repo root as context fails with COPY failed: file not found.

Impact: The Dockerfile uses COPY requirements.txt . which expects the file in the build context root.

Resolution: Use AIAgent/ as the build context:

az acr build --registry esiqnewacr --image esiqnew-agent:v1 --file AIAgent/Dockerfile AIAgent/

5. Classic ESIQ Resources

CLEANUP Old resources still running

Note: The classic Foundry resources in ESIQ-RG (swedencentral) are still provisioned. They cost approximately $5/month idle.

To clean up:

az group delete --name ESIQ-RG --yes --no-wait
